# edificio-marcellus

Aplicação para contato e envio de e-mail para imobiliarias.

Utilizando Nodejs, Angularjs 1 e outras frameworks

Banco de dados Mysql

Obs: Codigo não foi minificado.

Variáveis de ambiente:
- DB_DIALECT
- DB_HOST
- DB_DATABASE
- DB_USER
- DB_PASSWORD
- MAILER_HOST
- MAILER_PORT
- MAILER_USER
- MAILER_PASS
- MAILER_CIPHERS
